# Enterprise Knowledge Intelligence Platform (EKIP)

A production-grade platform that turns an organization's documents into a queryable, cited, multi-agent knowledge system — demonstrating advanced AI Engineering: RAG, GraphRAG, Agentic AI, guardrails, evaluation, and MLOps.

> Not another RAG chatbot. Every answer is retrieved hybrid (BM25 + dense + graph), reranked, generated by a domain agent, and verified for citations and groundedness before it reaches the user.

## Architecture

See [docs/02-architecture.md](docs/02-architecture.md) for full diagrams.

```
React/TS frontend ──> FastAPI (REST + WS) ──> Guardrails ──> Agent Orchestrator
                                                              │
                              Hybrid Retrieval (Qdrant · pgvector · BM25 · RRF · Rerank · Neo4j GraphRAG)
                                                              │
                              Async workers: ingest → OCR → clean → chunk → embed → graph
```

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, React Query, Zustand, Recharts |
| Backend | FastAPI, Pydantic, SQLAlchemy, Celery, WebSockets |
| Stores | PostgreSQL + pgvector, Qdrant, Neo4j, Redis, MinIO/S3 |
| AI | OpenAI / Anthropic / Cohere / Voyage / BGE / E5 (adapter pattern), cross-encoder reranking |
| Ops | Docker Compose, GitHub Actions, Prometheus/Grafana, OpenTelemetry |

## Getting Started (development)

```bash
cp .env.example .env       # fill in provider API keys
docker compose up -d       # postgres, qdrant, neo4j, redis, minio
# backend and frontend services are enabled as their milestones land
```
